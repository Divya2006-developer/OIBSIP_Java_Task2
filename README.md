OIBSIP_Java_Task2
ATM Interface System

Objective : 
Create a console-based ATM interface that simulates real banking operations including authentication, transactions, and account management.

Features : 
- Secure login (User ID & PIN authentication)
- Transaction History with timestamps
- Withdraw functionality with balance validation
- Deposit money
- Fund transfer between accounts
- Real-time balance updates
- Multi-screen navigation using CardLayout

Tools & Technologies Used : 
- Language: Java
- GUI Framework: Java Swing
- Text Editor : Visual Studio Code
- Components:CardLayout, JFrame, JPanel, JTextField, JPasswordField
- Date/Time:SimpleDateFormat
